# IPGeoLocation
===============

**Free, Open-Source Tool to retrieve IP Geolocation information**

Powered by [ip-api](http://ip-api.com/docs/)

IPGeoLocation is coded in Python 3.

Geolocation Information
---
* ASN
* City
* Country
* Country Code
* ISP
* Latitude
* Longtitude
* Organization
* Region Code
* Region Name
* Timezone
* Zip Code
